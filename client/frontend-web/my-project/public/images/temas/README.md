# Imágenes Temáticas para Misiones

## Estructura de Carpetas

Coloca las imágenes según el tema visual de cada misión:

```
public/images/temas/
├── medieval/     # Imágenes de castillos, espadas, escudos, caballeros
├── anime/        # Imágenes estilo anime/manga
├── espacial/     # Imágenes de naves, planetas, estrellas
├── fantasia/     # Imágenes mágicas, dragones, hechizos
├── ciencia/      # Imágenes de laboratorios, átomos, fórmulas
├── naturaleza/   # Imágenes de bosques, animales, plantas
├── urbano/       # Imágenes de ciudades, edificios modernos
├── oceano/       # Imágenes de océanos, peces, submarinos
└── default/      # Imágenes por defecto (educativas genéricas)
```

## Nombres de Archivos Recomendados

Para cada tema, puedes usar nombres como:
- `header-bg.jpg` o `header-bg.png` - Imagen de fondo para el header
- `icon.png` - Icono representativo del tema
- `card-bg.jpg` - Imagen de fondo para las tarjetas

## Formatos Soportados

- JPG/JPEG (recomendado para fotos)
- PNG (recomendado para iconos con transparencia)
- WebP (mejor compresión)

## Tamaños Recomendados

- Header: 1200x300px
- Card background: 800x600px
- Icon: 128x128px o 256x256px
